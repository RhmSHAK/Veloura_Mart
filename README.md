# 🛍️ Veloura Mart

Veloura Mart is a modern and responsive e-commerce frontend built with **React**, **Vite**, **Tailwind CSS**, and **DaisyUI**. The application provides a clean shopping experience with product browsing, category filtering, product details, and a shopping cart.

---

## 🚀 Live Demo

🔗 [https://your-vercel-link.vercel.app]

---

## 📌 Features

- Responsive UI for desktop, tablet, and mobile
- Modern Hero Banner
- Featured Products Section
- Product Listing
- Category Filtering
- Product Details Page
- Shopping Cart
- Dynamic Routing with React Router
- Beautiful UI using Tailwind CSS & DaisyUI
- Loading Component
- Custom 404 Not Found Page

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/Veloura_Mart.git
```

Go to the project directory

```bash
cd Veloura_Mart
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
